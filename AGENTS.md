# Repository Guidance

## GitHub Account and Remotes

Keep Git remotes on SSH for this repository. If push or fetch authentication appears to use the wrong GitHub account, do not switch the remote to HTTPS as a workaround.

Work on `faunt/slower.uk` and related Yearly repos should use the `faunt` GitHub account, not `DaveDesigner`. If SSH authenticates as `DaveDesigner`, stop and check whether the local SSH/GitHub account selection needs to be changed before pushing.
